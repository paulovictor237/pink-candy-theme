# Pink Candy — Zed Theme

A vivid dark theme with a striking **pink accent** (`#FF1277`), ported from the popular VS Code [Pink Candy](https://github.com/KubaP/vscode-pink-candy) theme by KubaP.

## Preview

| Element | Color |
|---|---|
| Background | `#22222A` |
| Accent / Cursor | `#FF1277` 🩷 |
| Keywords | `#D177F5` 💜 |
| Functions | `#10B1FE` 🔵 |
| Strings | `#3FC56B` 🟢 |
| Types | `#15C9C5` 🩵 |
| Numbers | `#F9C859` 🟡 |
| Comments | `#636D83` (italic) |

## Installation

Search for **Pink Candy** in the Zed Extensions store (`zed: extensions`).
